# pm-research-tools

产品经理调研工具集。提供日常市场调研、竞品分析、用户研究所需的常用工具集合。

## 使用场景

- 竞品动态追踪
- 行业信息收集
- 用户评价分析
- 市场数据获取
- 文档资料解析

## 核心工具

### 1. 🌐 多搜索引擎搜索

**支持引擎**：
- Bing（国际搜索）
- Google（需代理）
- 百度（中文搜索）
- 搜狗（中文搜索）
- DuckDuckGo（隐私搜索）

**搜索命令**：
```
搜索 [关键词] 渠道：Bing
搜索 [关键词] 渠道：知乎
搜索 [关键词] 渠道：小红书
搜索 [关键词] 渠道：微博
```

### 2. 📄 网页内容抓取

**功能**：
- 提取网页正文内容
- 转换为 Markdown 格式
- 提取标题/作者/发布时间
- 提取图片和链接

**支持平台**：
- 知乎文章/回答
- 小红书笔记
- 微博帖子
- 微信公众号
- 36kr/虎嗅等科技媒体
- 行业报告网站

### 3. 📎 文档解析

**支持格式**：
- PDF（报告/文档）
- Word（.doc/.docx）
- Excel（.xlsx/.xls）
- PPT（.ppt/.pptx）
- Markdown（.md）

**提取内容**：
- 文本内容
- 表格数据
- 图片（需处理）
- 文档结构

### 4. 📊 社交平台内容提取

**支持平台**：

| 平台 | 内容类型 | 提取字段 |
|------|---------|---------|
| 知乎 | 文章/回答/评论 | 标题/作者/点赞/内容 |
| 小红书 | 笔记/评论 | 标题/作者/收藏/内容 |
| 微博 | 微博/评论 | 内容/转发/评论/作者 |
| 微信公众号 | 文章 | 标题/作者/阅读量/内容 |
| B站 | 视频/评论 | 标题/UP主/播放/弹幕 |

## 工作流程

### 调研任务流程

```
Step 1: 明确调研目标
├── 确定调研主题
├── 列出关键问题
└── 规划信息来源

Step 2: 执行搜索
├── 多引擎并发搜索
├── 关键词组合搜索
└── 竞品名称搜索

Step 3: 内容抓取
├── 抓取高价值页面
├── 提取正文内容
└── 过滤广告/无关内容

Step 4: 文档解析
├── 收集相关报告/PDF
├── 解析文档内容
└── 提取关键数据

Step 5: 信息整理
├── 按主题分类
├── 去重和筛选
├── 提取关键洞察
└── 整理成调研报告
```

### 典型调研任务

#### 竞品分析调研
```
1. 搜索：竞品A 竞品B 竞品C
2. 抓取：各竞品官网/产品页面
3. 社交：知乎/小红书用户评价
4. 整理：竞品对比表
```

#### 行业趋势调研
```
1. 搜索：行业关键词 + 趋势/报告
2. 抓取：行业报告/媒体文章
3. 解析：下载的行业PDF报告
4. 整理：趋势洞察总结
```

#### 用户反馈调研
```
1. 搜索：产品相关用户反馈
2. 抓取：应用商店评价
3. 社交：小红书/微博用户讨论
4. 整理：用户痛点清单
```

## 工具使用指南

### 搜索语法

```markdown
# 基础搜索
搜索：产品经理工具

# 限定平台
搜索：竞品分析 site:zhihu.com
搜索：小红书种草 某产品

# 限定时间
搜索：2024年 行业报告
搜索：最近一个月 用户评价

# 多关键词
搜索：A AND B OR C
搜索：竞品 对比 评测
```

### 内容抓取命令

```markdown
# 抓取单个页面
抓取：https://example.com/article

# 抓取并转换
抓取并转为Markdown：https://example.com/article

# 批量抓取（同一域名）
抓取该页面所有文章：https://example.com/blog
```

### 文档解析命令

```markdown
# 解析本地文件
解析PDF：~/Downloads/report.pdf
解析Word：~/Documents/requirements.docx

# 解析并提取
提取表格数据：~/Documents/data.xlsx
提取图片：~/Documents/report.pdf
```

## 注意事项

1. **版权合规**：抓取内容仅供个人学习研究使用

2. **时效性**：标注信息来源时间，优先最新内容

3. **交叉验证**：重要数据需多个来源确认

4. **去广告**：抓取时过滤广告和无用内容

5. **数据脱敏**：用户评价等注意个人信息保护

## 验证方式

- 调研报告包含完整的搜索/抓取/解析过程记录
- 有具体的数据支撑（链接/截图/文档摘录）
- 有关键洞察提炼，不是简单的信息堆砌
- 可交付到飞书云文档
